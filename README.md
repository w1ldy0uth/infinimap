# InfiniMap

InfiniMap is a desktop C++ application for generating and editing detailed multi-layer maps of fictional worlds.

## Current status

- MVP planning is in progress.
- Scope freeze is defined and tracked in GitHub.
- Initial CMake project scaffold is present.

## MVP focus

The MVP targets:

1. Procedural 2D world generation (seeded and deterministic).
2. Core geography realism (terrain, coastlines, hydrology, biomes).
3. Basic map editing tools (landmass, rivers, biomes).
4. Native project save/load (`.infm`) and raster export (`.jpeg`).
